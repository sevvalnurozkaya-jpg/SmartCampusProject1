# 🎓 Smart Campus Management System

This project is a web-based Smart Campus Management System developed using Django. It allows students, staff, and administrators to manage campus activities efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Panel
- View and apply for events
- Make reservations (study rooms, sports fields, equipment)
- Join clubs
- Send feedback
- Apply for staff role

### 🧑‍💼 Staff Panel
- Create and manage events
- View and approve student applications
- Manage reservations
- Post announcements
- View analytics dashboard (charts & reports)

### 🛠️ Admin Panel
- Manage users and roles
- Approve/reject events
- Review staff applications
- View system overview

---

## 📊 Analytics Dashboard
- Most Popular Events (Bar Chart)
- Most Popular Clubs (Pie Chart)
- Most Active Students (Line Chart)

---

## 🧱 Technologies Used
- Python
- Django
- SQLite
- HTML / CSS
- JavaScript (Chart.js)
